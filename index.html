Hola, soy Fabian Villarroel

¿Quién Soy?

Soy un apasionado desarrollador web frontend con experiencia en HTML, CSS y Bootstrap. Residente de Venezuela, me encanta sumergirme en el mundo de los videojuegos y encontrar inspiración para mis proyectos. Cuando no estoy frente a la pantalla, me dedico a escribir y compartir mis historias en Wattpad. Combinando mi creatividad con mis habilidades técnicas, busco crear experiencias web únicas y envolventes.

Desarrollo Web

Experiencia en el desarrollo de interfaces de usuario atractivas y funcionales utilizando HTML, CSS y Bootstrap. Mis habilidades incluyen maquetación, diseño responsive y la implementación de componentes prediseñados..
